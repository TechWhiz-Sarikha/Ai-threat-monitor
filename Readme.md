## в¬Ў Overview

A Python-powered security engine that **detects OWASP Top 10 web attacks in real time**, classifies traffic using **Machine Learning**, scores severity, and generates **SOC-analyst-style AI incident summaries** вҖ” all visualized on a live React dashboard built with a dark terminal aesthetic.

Built to demonstrate practical skills in:
`Python` В· `Security Automation` В· `Machine Learning` В· `REST APIs` В· `OWASP` В· `Frontend Engineering`

---

## рҹҺҜ Features

| Module | What It Does |
|---|---|
| рҹӨ– **ML Classifier** | TF-IDF + Logistic Regression pipeline classifies every payload as `malicious` or `normal` with confidence score |
| рҹ”Қ **Threat Detection Engine** | Regex-based OWASP rule engine catches SQLi, XSS, Directory Traversal, Command Injection |
| вҡ пёҸ **Severity Scoring** | Maps each attack to `Critical` / `High` / `Medium` / `Low` severity level |
| рҹ§  **AI Summary Engine** | Generates structured SOC-style incident reports: what happened, impact, recommended action, OWASP reference |
| рҹ“Ӯ **Log File Analysis** | Upload any `.txt` or `.log` file вҖ” system analyzes every line and returns full threat report |
| рҹ“Ҡ **Live Dashboard** | React + Vite dashboard with real-time stats, filterable threat table, attack distribution, and click-to-expand incident detail panel |
| рҹ”„ **Auto Refresh** | Dashboard polls the API every 30 seconds automatically |


